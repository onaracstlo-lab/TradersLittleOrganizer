TLO Source and Utilities Bundle v336

Public application version: v1.2 Build 336

Build 336 summary
- Standalone tlo-tag and the GUI Tag window now always tag the selected path directly.
- Inventory-only Tag in Place, Tag Copy, and Tag Copy Destination controls are no longer accepted or used by tlo-tag.
- Inventory and Add Shows retain their existing tagging/copy controls.
- Standalone Tag does not copy or delete content. Verified SHN-to-FLAC conversion remains the sole exception: the source SHN/SHNF is deleted only after successful replacement.
- --myTLO remains accepted for compatibility but is hidden from help and user documentation.

- TLO_Inventory_User_Manual_v336.rtf: current end-user manual.
- TLO_Inventory_Requirements_Working_v336.docx: current requirements/development document.
