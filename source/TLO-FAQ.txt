Q:Is TLO a commercial application:
A: No, TLO is freeware.

Q: Does TLO do a good job of inventory and tagging?
A: TLO does a better job of identifying show metadata than LiveShowTagger.

Q: Do I need to define TLOHome before I do anything?
A: Yes, TLOHome is where all the internal files live. It can be set as an environment variable, on the command line or in the GUI. It can be added to the command line of a shortcut in Windows.

Q: Is there anything else I need to do, other than define TLOHome, before I start?
A: No, there are no hard requirements.  However, you might want to set an environment variable for the drive where you're currently storing new downloads, and get a setlist.fm user key and set the environment variable for that, too.

Q: How difficult is it to set the TLO-related environment variables?
Q: Not very. It's all described in sections 3 and 4 of the User Manual.

Q: Can TLO ever do anything destructive?
A: Yes!!! If you compliantly rename the original folders, do a tag in place, or a copy and delete, then the originals will not be the same (or be deleted, as the action implies) and this is not reversible. However, these actions are all user controlled and off by default. You never lose your content, but if you want to keep a pristine copy for reseeding, these actions (except the compliant rename) will prevent reseeding.

Q: What if I forget and accidentally re-inventory a drive?
A: It will replace the current inventory of that drive, assuming the volume has a name. If there's no volume name, it will create duplicate entries for the content on that drive.
