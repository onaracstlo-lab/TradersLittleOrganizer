Traders Little Organizer is a collection of apps, written in Python by ChatGPT, that can be used to inventory a music collection, search across the inventory
and tag flac files with artist, album, track number and song title based on collecting metadata about the show from the directory structure, setlist files, existing tags, etreeDB.org, and setlist.fm. EtreeDB.org and setlist.fm are accessed 
to provide venue, location and setlist info.

There is an artist database that comes from a scrape of artists in etreeDB and a venue list.

An excellent replacement for LiveShowTagger - works in bulk, more discriminating about shoosing setlist files, and less finicky about song numbering.
