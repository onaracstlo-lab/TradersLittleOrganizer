Traders Little Organizer is a collection of apps, written in Python by ChatGPT, that will inventory a music collection, provides a search across the inventory
and provides a means to tag .flac files with artist, album, track number and song title based on the metadata collected. EtreeDB.org and setlist.fm are accessed 
to provide venue, location and setlist info.

There is an artist database that comes from a scrape of artists in etreeDB and a venue list.
